# 📄 Write employee onboarding guide for first-time USDC wage claimants

**Labels**: `documentation`, `good first issue`, `help wanted`
**Complexity**: 🟢 Beginner — no blockchain knowledge required

## Summary
New employees receiving wages in USDC for the first time need a plain-language guide explaining how to claim their wages using Freighter wallet. Most workers in the target corridors (Nigeria, Kenya, Ghana, Philippines) have never used a crypto wallet before.

## What needs to be done
Create `docs/guides/employee-onboarding.md` covering:

1. What is a Stellar wallet? (2–3 sentences, no jargon)
2. How to install Freighter browser extension (with screenshots)
3. How to create a Stellar wallet address
4. How to fund the wallet with at least 1 XLM (for account activation)
5. How to navigate to `/dashboard` and claim wages
6. How to check your balance after claiming
7. FAQ section (minimum 5 questions employees actually ask)

## Acceptance criteria
- [ ] Written in plain English at roughly an 8th-grade reading level
- [ ] Includes Freighter installation steps with public screenshots (link to official Freighter docs/screenshots, do not upload screenshots yourself)
- [ ] FAQ addresses: "What if I lose my wallet?", "Is USDC real money?", "How long does it take?", "What are the fees?", "Can I convert to local currency?"
- [ ] Reviewed by at least one non-technical person before PR submission
- [ ] Placed at `docs/guides/employee-onboarding.md`

## Notes
Do NOT explain Soroban smart contracts or blockchain internals — this guide is for workers, not developers. The tone should feel like a friendly WhatsApp message, not a technical manual.
